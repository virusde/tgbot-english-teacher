# Telegram English Teacher Bot

Простой Telegram-бот для изучения английского языка:

- выдает короткие уроки,
- после урока сразу дает упражнения по теме,
- разбирает ошибки после ответов,
- предлагает выбрать: практиковаться еще или идти дальше,
- умеет генерировать новые темы и гибко проверять ответы через OpenAI API,
- показывает слова текущей темы,
- проводит общий квиз по переводу,
- сохраняет прогресс пользователя в локальный JSON,
- подсказывает слова, которые стоит повторить.

## Запуск

1. Создайте бота через BotFather и получите токен.
2. Если у вас есть свой Python, установите зависимости:

```bash
pip install -r requirements.txt
```

4. Создайте `.env` на основе `.env.example` и укажите:

```env
BOT_TOKEN=ваш_токен
OPENAI_API_KEY=ваш_openai_api_key
OPENAI_MODEL=gpt-5-mini
OPENAI_LESSON_MODEL=gpt-5-mini
OPENAI_REVIEW_MODEL=gpt-5-nano
OPENAI_TIMEOUT_SECONDS=90
```

Если `OPENAI_API_KEY` не задан, бот продолжит работать на встроенных уроках и локальной проверке ответов.

5. Запустите бота любым из способов:

```bash
python main.py
```

или из PowerShell:

```powershell
.\run_bot.ps1
```

Скрипт `run_bot.ps1` сначала попробует локальный portable Python в `tools/python312`, если он есть, а иначе использует системный `python`.

Если PowerShell блокирует запуск `.ps1` из-за Execution Policy, используйте:

```cmd
run_bot.cmd
```

или явно через локальный интерпретатор проекта:

```powershell
.\tools\python312\python.exe .\main.py
```

## Тесты

Для быстрого регрессионного прогона используйте:

```powershell
.\tools\python312\python.exe -m unittest discover -s tests -v
```

Тесты покрывают встроенные уроки, хранение прогресса и чистую внутреннюю логику без реальных запросов к Telegram или OpenAI.

## Команды

- `/start` — приветствие и меню
- `/lesson` — бот спросит тему или предложит выбрать случайную
- `/lesson путешествия` — сразу урок на нужную тему
- `/topic <тема>` — задать тему следующего урока
- `/words` — слова из текущего урока
- `/quiz` — общий мини-тест по изученным словам
- `/repeat` — слабые слова на повторение
- `/progress` — статистика
- `/help` — справка

## OpenAI режим

Если в `.env` указан `OPENAI_API_KEY`, бот:

- генерирует новые темы уроков через OpenAI,
- старается не повторять недавние темы и сценарии,
- умеет принять тему урока от пользователя,
- подбирает упражнения под тему,
- оценивает ответы мягче, чем простое точное совпадение,
- объясняет ошибки более естественно.

Можно либо использовать `/topic путешествия`, либо просто написать боту фразу вроде:

- `Хочу урок про путешествия`
- `Хочу урок про собеседование`
- или вызвать `/lesson`, после чего бот сам спросит тему

Если тему не задавать, бот выбирает ее сам.

По умолчанию используется модель `gpt-5-mini`, но ее можно поменять через `OPENAI_MODEL`.
Для более экономного режима можно отдельно указать:

- `OPENAI_LESSON_MODEL=gpt-5-mini` для генерации уроков
- `OPENAI_REVIEW_MODEL=gpt-5-nano` для проверки коротких ответов
- `OPENAI_TIMEOUT_SECONDS=90` если хотите дать модели больше времени на ответ

## Структура

- `main.py` — точка входа
- `app/bot.py` — хендлеры Telegram
- `app/content.py` — уроки и словарь
- `app/storage.py` — хранение прогресса
- `data/progress.json` — создается автоматически
- `tools/python312` — локальный portable Python, если вы решили хранить его вне Git
